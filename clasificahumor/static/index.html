¿El tweet pretende ser humorístico?

El fin de esta página es armar una base de datos de opiniones sobre chistes para en el futuro poder hacer investigación sobre Humor Computacional.

Por ejemplo, puede usarse para predecir si un tweet es humorístico o no, qué tan gracioso le puede resultar a la gente o inclusive para generar humor :O

La página fue originalmente creada en 2014 por dos estudiantes del proyecto de grado pgHumor (Santiago Castro y Matías Cubero) de Ingeniería en Computación – Universidad de la República, Uruguay, apoyados por la ANII, y en 2018 se volvió a lanzar para conseguir más votos :D

Gracias a tu ayuda, vamos a poder obtener una base de datos de buena calidad. ¡¡¡MUCHAS GRACIAS!!!
